Usability Testing Report

Goal of the site

The main goal of the website is to attract more customers to petite treats bakery by getting them online. This is especially aimed at the younger, more tech savvy crowd.

User Testing Details

  • Name: Anu
  • Location: India
  • Age: 20
  • Employment: Student

Pre Test Questions

  • What do you expect from a bakery site? A: I would like information about the products wasily and be able to order online.
  • Do you think current bakery websites or general websites for stores are good? A: It depends, some of them are really good while others are ugly or cluttered and sometimes even broken.
  • Will you be willing to sign up for a newsletter to get discounts? A: Yes, I love discounts and to always be aware when there is one going on will be great.

Test Tasks and Schedule

  • Click on the link to the website, which will take you to the home page
  • Mess around, try menu items, look at the specials, etc.
  • Try to sign up for the newsletter
  • Do whatever else you want and see if everything works and is to your liking

Post test debrief

  • What did you think about the website? A: Everything worked and I was able to get all the information that I would like quite easily.
  • Did you like the design of the website? A: It was fine, it wasn't very fancy but it was easy on the eyes and I could navigate easily.
  • Was the website usable on a phone? A: Yes, I didn't have any problem using it on the phone.

Usability Scale

  • Learnability: The site was easy to use and navigation made sense. - 9.5/10
  • Efficiency: After initial learning curve the user can do things very easily. - 8/10
  • Memorability: The design is easy to use and remember, the user won't have any problem using the site after a long time. - 9/10
  • Errors: The site was error free. - 10/10
  • Satisfaction: The site was good and easy to use. - 9/10
